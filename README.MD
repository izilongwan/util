### @izilong/util

- 安装｜install
```shell
npm i @izilong/util
```
```shell
yarn add @izilong/util
```

- 使用｜use in ES6
```js
import util from '@izilong/util'
```

- NODE
```js
const util = require('@izilong/util')
```
