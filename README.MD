# @izilong/util

Install
```shell
$ npm i @izilong/util
$ or yarn add @izilong/util
```

## Use

In ESM.js:
```js
import * as util from '@izilong/util'
or import {
  array,
  async,
  bom,
  data,
  date,
  element,
  geometric,
  object,
  plugins,
  regexp,
  string,
  url,
  xhr,
  _function } from '@izilong/util'
```

In Node.js:
```js
const util = require('@izilong/util')
or const {
  array,
  async,
  bom,
  data,
  date,
  element,
  geometric,
  object,
  plugins,
  regexp,
  string,
  url,
  xhr,
  _function } = require('@izilong/util')
```
